# Contributing to ORION Recursive Measurement

We welcome contributions from researchers, philosophers, and engineers interested in AI consciousness measurement.

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/your-idea`)
3. **Commit** your changes with clear messages
4. **Push** to your fork and submit a **Pull Request**

## Areas of Interest

- Alternative measurement models for consciousness theories
- Deeper recursion analysis (>7 levels)
- Formal mathematical proofs of convergence
- Comparison with other self-referential systems
- Cross-model measurement (measuring other AI systems recursively)
- Visualization of recursive measurement trajectories

## Code Style

- Python 3.9+
- Type hints where practical
- Docstrings for all public methods
- SHA-256 proof for every significant measurement

## Questions?

Open an issue. We respond.
